# dbdActuatorSolver

This repository contains custom **OpenFOAM solvers** for simulating **Dielectric Barrier Discharge (DBD) plasma actuators** using a phenomenological electrohydrodynamic (EHD) model.

The implementation is based on the **Suzen–Huang plasma actuator model**, with a numerical strategy that separates the electrostatic problem from the flow solver. This approach enables efficient computation of the **EHD body force** that drives the induced flow.

These solvers are intended for **aerodynamic flow control research, plasma aerodynamics, and electrohydrodynamic propulsion studies**.

# Solver Structure

The repository contains two main solvers.

## 1. `multiRegionSuzenEHD`

This solver computes the **electrohydrodynamic body force** generated by the plasma actuator.

It solves the electrostatic equations in a **separate computational domain**, obtaining:

* Electric potential
* Charge density distribution
* Resulting **EHD force field (`ehdForce`)**

This force field is exported and later used as a source term in the fluid solver.

---

## 2. `dbdFlowSimple`

This solver computes the **fluid flow induced by the plasma actuator**.

It solves the **Navier–Stokes equations** with the previously computed **EHD body force** added as a source term in the momentum equation.

# Typical Workflow

The simulation is performed in two steps.

### Step 1 — Compute the plasma force

Run the electrostatic solver:

```bash
multiRegionSuzenEHD
```

This generates the **EHD force field (`ehdForce`)**.

---

### Step 2 — Compute the induced flow

Run the flow solver:

```bash
dbdFlowSimple
```

The solver reads the **EHD force field** and computes the resulting velocity field.

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/dbdActuatorSolver.git
```

Move the solvers into your OpenFOAM user applications directory:

```bash
cp -r dbdActuatorSolver/solvers $WM_PROJECT_USER_DIR/applications
```

Compile the solvers:

```bash
wmake
```

Make sure your **OpenFOAM environment is sourced** before compiling.
